Приложение A: Формальные типы настройки

Следующие таблицы содержат примеры доступных формальных типов настраиваемых модулей:

Формальный тип

Фактический тип

Неполный тип

Формат: type T;

Любой тип

Дискретный тип

Формат: type T is (<>);

Любой целочисленный тип, модульный тип или тип перечисления

Тип диапазона

Формат: type T is range <>;

Любой целочисленный тип со знаком

Модульный тип

Формат: type T is mod <>;

Любой модульный тип

Тип с плавающей запятой

Формат: type T is digits <>;

Любой тип с плавающей запятой

Двоичный тип с фиксированной запятой

Формат: type T is delta <>;

Любой двоичный тип с фиксированной точкой

Десятичный тип с фиксированной запятой

Формат: type T is delta <> digits <>;

Любой десятичный тип с фиксированной запятой

Определенный неограниченный частный тип

Формат: type T is private;

Любой неограниченный, определенный тип

Неограниченный частный тип с дискриминантом

Формат: type T (D : DT) is private;

Любой неограниченный тип с дискриминантом

Тип доступа

Формат: type A is access T;

Любой тип доступа для типа T

Определенный производный тип

Формат: type T is new B;

Любой конкретный тип, полученный из базового типа B

Ограниченный частный тип

Формат: type T is limited private;

Любой определенный тип, ограниченный или нет

Неполный тип тега

Формат: type T is tagged;

Любой конкретный, определенный, тегированный тип

Определенный частный тип с тегами

Формат: type T is tagged private;

Любой конкретный, определенный, тегированный тип

Определенный ограниченный частный тип с тегами

Формат: type T is tagged limited private;

Любой конкретный определенный тип с тегом , ограниченный или нет.

Определенный абстрактный частный тип с тегами

Формат: type T is abstract tagged private;

Любой неограниченный, определенный тип с тегом , абстрактный или конкретный

Определенный абстрактный ограниченный частный тип с тегами

Формат: type T is abstract tagged limited private;

Любой определенный тип с тегами, ограниченный или нет, абстрактный или конкретный

Определенный производный тип с тегами

Формат: type T is new B with private;

Любой конкретный тип с тегами, полученный из базового типа B

Определенный абстрактный производный тип с тегами

Формат: type T is abstract new B with private;

Любой тип с тегом, полученный из базового типа B или конкретного базового типа B

Тип массива

Формат: type A is array (R) of T;

Любой тип массива с диапазоном R, содержащий элементы типа T

Тип интерфейса

Формат: type T is interface;

Любой тип интерфейса T

Ограниченный тип интерфейса

Формат: type T is limited interface;

Любой ограниченный тип интерфейса T

Тип интерфейса задачи

Формат: type T is task interface;

Любой тип интерфейса задачи T

Тип синхронизированного интерфейса

Формат: type T is synchronized interface;

Любой синхронизированный интерфейс типа T

Тип защищенного интерфейса

Формат: type T is protected interface;

Любой защищенный интерфейс типа T

Производный тип интерфейса

Формат: type T is new B and I with private;

Любой тип T, полученный из базового типа B и интерфейса I

Производный тип с несколькими интерфейсами

Формат: type T is new B and I1 and I2 with private;

Любой тип T, полученный из базового типа B и интерфейсов I1 и I2

Абстрактный производный тип интерфейса

Формат: type T is abstract new B and I with private;

Любой тип T, полученный из абстрактного базового типа B и интерфейса I

Ограниченный производный тип интерфейса

Формат: type T is limited new B and I with private;

Любой тип T, полученный из ограниченного базового типа B и ограниченного интерфейса

Абстрактный ограниченный производный тип интерфейса

Формат: type T is abstract limited new B and I with private;

Любой тип T, полученный из абстрактного огранич. енного базового типа B и ограниченного интерфейса

Тип синхронизированного интерфейса

Формат: type T is synchronized new SI with private;

Любой тип T, полученный из синхронизированного интерфейса SI

Абстрактный синхронизированный тип интерфейса

Формат: type T is abstract synchronized new SI with private;

Любой тип T, полученный из синхронизированного интерфейса SI

Неограниченная версия типа

Многие из приведенных выше примеров могут быть использованы для формальных неопределенных (неограниченных) типов:

Формальный тип

Фактический тип

Неопределенный неполный тип

Формат: type T (<>);

Любой тип

Неопределенный неограниченный частный тип

Формат: type T (<>) is private;

Любой неограниченный тип неопределенный или определенный

Неопределенный ограниченный частный тип

Формат: type T (<>) is limited private;

Любой тип, ограниченный или нет, неопределенный или определенный

Неполный неопределенный частный тип с тегами

Формат: type T (<>) is tagged;

Любой конкретный тип с тегами, неопределенный или определенный

Неопределенный частный тип с тегами

Формат: type T (<>) is tagged private;

Любой конкретный, ограниченный тип с тегами, неопределенный или определенный

Неопределенный ограниченный частный тип с тегами

Формат: type T (<>) is tagged limited private;

Любой конкретный тип с тегами, ограниченный или нет, неопределенный или определенный

Неопределенный абстрактный частный тип с тегами

Формат: type T (<>) is abstract tagged private;

Любой неограниченный тип с тегами, неопределенный или определенный, абстрактный или конкретный

Неопределенный абстрактный ограниченный частный тип с тегами

Формат: type T (<>) is abstract tagged limited private;

Любой тип с тегами, ограниченный или нет, неопределенный или определенный,абстрактный или конкретный

Неопределенный производный тип с тегами

Формат: type T (<>) is new B with private;

Любой тип с тегами, производный от базового типа B, неопределенный или определенный

Неопределенный абстрактный производный тип с тегами

Формат: type T (<>) is abstract new B with private;

Любой тип с тегами, производный от базового типа B, неопределенный или определенный абстрактный или конкретный

Те же примеры могут также содержать дискриминанты. В этом случае, (<>) заменяется списком дискриминантов, например: (D: DT).
